Add contact information to the top of your website.

== Description ==

Customize color, appearance, and add HTML to the top of your website. Choose whether or not to include a call to action button as well.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `contact-info-bar.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Inside WordPress admin, go to Tools > Contact Info Bar
